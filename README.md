Overview

This project focuses on estimating how well a student might perform academically based on a few everyday factors. The idea behind the work is simple — students with better study habits and healthier routines generally score higher. By collecting a small dataset and applying machine learning methods, the project builds a model that can guess the final marks of a student.

The main intention behind the project is to show how data and simple algorithms can be used to understand patterns in learning behavior. It is also a practical way to explore how predictive systems are built using Python.


---

Features

Uses learning-related attributes such as study duration, attendance, sleep schedule, and internet usage.

Trains two different models and compares their performance.

Shows visual plots to understand the model’s predictions.

Easy to run on any system with Python installed.

Clean and beginner-friendly structure.

---

How to Install and Run the Project

1. Make sure Python is installed on your system.


2. Open a terminal or command prompt inside the project folder.


3. Install the required packages:

pip install scikit-learn pandas numpy matplotlib

4. Place the dataset file (student_performance.csv) in the main folder.


5. Run the notebook or Python script:

For notebook:

jupyter notebook

Open the .ipynb file and run all cells.

For Python script:

python model.py
